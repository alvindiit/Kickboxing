# JA Boxing - Kickboxing Landing Page

A modern, responsive landing page for a kickboxing gym, inspired by professional boxing websites.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - Scroll animations and hover effects for better user engagement
- **Modern UI/UX** - Clean, professional design with bold typography
- **Sections Include:**
  - Hero section with call-to-action
  - About section with statistics
  - Classes overview with three different programs
  - Weekly class schedule table
  - Contact form and information
  - Responsive navigation with mobile menu

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Montserrat)

## How to Use

1. Open `index.html` in your web browser
2. Navigate through different sections using the menu
3. Customize the content, colors, and images to match your brand

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #e63946;
    --secondary-color: #1d3557;
    --accent-color: #f1faee;
}
```

### Content
- Update text in `index.html`
- Replace placeholder emojis with actual images
- Modify class schedule times and days
- Update contact information

### Images
Replace the placeholder emoji icons with actual images:
- Add images to an `images` folder
- Update the image sources in the HTML
- Adjust the CSS for proper image sizing
